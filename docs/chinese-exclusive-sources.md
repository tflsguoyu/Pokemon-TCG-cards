# Simplified Chinese Exclusive Card Sources

Reference links for researching Simplified Chinese exclusive Pokemon TCG cards.

Use these as starting points when adding Chinese-exclusive cards to this project. They are external collector-maintained lists, so verify each candidate against the project's data rules before importing:

- The card should be an official physical Pokemon TCG card.
- Prefer TCGdex metadata when available.
- Existing project image rules still apply unless the collection policy is changed.
- Confirm whether the card is truly exclusive artwork, or only a Simplified Chinese localized release.
